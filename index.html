☁️ First-Time Cloud Setup

This is a one-time setup. After this, you will share the updated HTML file with all your PCs — they just open it and log in. No setup needed on other PCs.

Step 1 — Create Free Supabase Account (2 min)

Go to supabase.com → Click Start your project → Sign up free with Google or email. No credit card. Free forever.

Step 2 — Create a New Project

Click New Project → Name it anything (e.g. mycompany) → Choose any region → Set a password → Wait ~2 minutes for it to start.
